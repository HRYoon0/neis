# NEIS 종합의견 자동입력 도우미

엑셀·CSV 파일에 정리한 내용을 나이스(NEIS)의 **학기말종합의견·행동특성및종합의견·창의적체험활동**
화면에 번호·이름으로 자동 입력하는 **크롬 확장 프로그램**입니다.

## 저장소 구성

| 경로 | 설명 |
| --- | --- |
| `neis-autofill/` | 확장 소스 (개발용 · 연습 페이지/심화진단 포함) |
| `dist/neis-autofill/` | 웹스토어 배포본 (개발용 요소 제거된 클린 빌드) |
| `dist/neis-autofill-1.0.2.zip` | 웹스토어 업로드 패키지 |
| `store/` | 스토어 등록 자료 (설명·스크린샷·개인정보처리방침) |
| `build.js` | 배포본 재생성 스크립트 (`node build.js`) |

## 개인정보처리방침

[store/PRIVACY.html](store/PRIVACY.html) — 어떤 개인정보도 수집·전송하지 않으며, 모든 처리는
사용자 브라우저 안에서만 이루어집니다.

## 설치 (개발자 모드)

자세한 사용법은 [neis-autofill/README.md](neis-autofill/README.md) 참고.
